# Flask Multi-Page Website

A simple Flask web application that demonstrates routing between multiple pages using HTML templates.

## Features

- Home page
- About page
- Flask routing
- HTML templates
- Jinja2 template rendering
- Beginner-friendly project


## Project Structure

```text
flask-multi-page-website/
│── app.py
│── requirements.txt
│── README.md
│── .gitignore
└── templates/
    ├── home.html
    └── about.html
```

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/flask-multi-page-website.git
```

2. Move into the project directory:

```bash
cd flask-multi-page-website
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python app.py
```

5. Open your browser and visit:

```
http://127.0.0.1:5000/
```
